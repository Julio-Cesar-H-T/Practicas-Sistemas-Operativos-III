Este repositorio centraliza todas las prácticas de laboratorio realizadas en la materia **Sistemas Operativos III**, enfocada en la administración avanzada de servidores Linux. Cada módulo está documentado en su propia carpeta con explicación de comandos, configuración aplicada y resultados obtenidos.

## 🧰 Tecnologías y Herramientas Cubiertas

A lo largo de los módulos se trabajó con:

- Administración de usuarios, grupos y permisos
- Gestión de paquetes y automatización de tareas (cron)
- Administración de discos y almacenamiento
- Shell scripting y manejo de SSH
- Servicios de red (HTTP, correo, impresión)
- Alta disponibilidad y clustering (Rsync, clusters HA)
- Hardening y seguridad en Linux (GPG2, firewall, IDS, 2FA)
- Active Directory / Samba / NFS
- Contenerización (Docker, Docker Compose)
- Infraestructura como Código (Terraform/Ansible)
